# test API REST con NODE JS para web developer backend CONDOR LABS

### GET `/api/items` 
Obtener todos los items

### PUT  `/api/items`
Crear un nuevo item

### GET `/api/items/:itemId`
Obtener un item por _id

### PUT `/api/items/:itemId`
Actualizar item

### DELETE `/api/items/:itemId`
Eliminar un item

Instalar
`npm install`

Ejecutar
`node app.js`
